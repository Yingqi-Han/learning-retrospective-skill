# Similar Projects

This skill is intentionally small and portable. It overlaps with, but does not replace, broader agent memory systems.

## Comparable Directions

- Claudeception: turns non-obvious Claude Code discoveries into skills and can use hooks for activation.
- 10x / agent-loom: a broader engineering discipline that records decisions, evidence, knowledge, tickets, and retrospectives.
- claude-memory-skill: a low-friction markdown memory protocol for Claude Code.
- agentmemory: a cross-agent memory service with hooks, indexing, and injection.

Useful starting points:

- https://github.com/blader/Claudeception
- https://github.com/z3z1ma/agent-loom
- https://github.com/hanfang/claude-memory-skill
- https://github.com/jayzeng/agentmemory

## Differentiation

Learning Retrospective focuses on one narrow moment:

- detect a repeated retry loop
- stop broad fallback behavior
- choose one evidence-backed next action
- write one small verified lesson to the right surface
- optionally ask any independent reviewer, not a specific model

It should compose with larger memory systems instead of competing with them.
